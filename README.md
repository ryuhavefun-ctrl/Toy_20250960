# 캠퍼스 머니 플래너 (Campus Money Planner)

대학생의 일상 소비를 쉽고 빠르게 기록할 수 있도록 만든 초간단 소비 관리 웹앱입니다.  
경영학부에서 배우는 *예산 관리, 비용 통제, 재무적 의사결정* 개념을 실제 생활에 적용해볼 수 있도록 기획되었습니다.


## 프로젝트 소개

대학생들은 학식·카페·배달·교통·도서·대외활동 등 다양한 지출을 경험하지만,  
은행 앱이나 기존 가계부 앱은 너무 기능이 많아 정작 나의 소비 습관을 파악하기 어렵다는 문제가 있습니다.

이 웹앱은 “간단함”에 초점을 맞추어,  
기록 → 확인 → 습관 인식의 흐름을 자연스럽게 경험할 수 있는 도구로 설계되었습니다.

- 따로 로그인 없음  
- 광고 없음  
- 입력 후 바로 확인  
- 저장은 브라우저(LocalStorage)에서만 진행


## ⚙️ 구현한 기능 (Features)

### ✔️ 1. 지출 입력 기능
- 날짜, 항목명, 카테고리, 금액 입력
- 입력 즉시 리스트에 표시

### ✔️ 2. 지출 리스트 표시
- 표 형태로 날짜/항목/카테고리/금액/삭제 버튼 표시

### ✔️ 3. 카테고리 필터
- 전체 / 식비 / 교통 / 쇼핑 / 자기계발 / 기타
- 선택된 카테고리만 필터링해서 리스트 표시

### ✔️ 4. 요약 정보 제공
- 현재 필터 기준 총 지출 금액
- 지출 건수 표시

### ✔️ 5. 삭제 기능
- 각 입력 항목을 개별적으로 삭제 가능

### ✔️ 6. localStorage 기반 데이터 저장
- 브라우저에 저장되어 새로고침해도 기록이 유지됨

## 🚀 실행 방법 (How to Run)

### ✔️ 로컬 환경에서 실행
1. 이 저장소를 다운로드하거나 Clone 합니다.
2. 폴더 안의 **index.html** 파일을 더블클릭합니다.
3. 브라우저(Chrome 등)에서 웹앱이 자동 실행됩니다.

### ✔️ GitHub Pages로 실행 (선택)
만약 GitHub Pages를 활성화했다면 아래 주소에서도 접근 가능합니다:
https://yourname.github.io/campus-money-planner/

## 🧪 기타 참고사항

- 모든 데이터는 로컬 저장소(localStorage)에만 저장됩니다.  
  → 서버로 전송되지 않아 개인정보 위험 없음
- PC·노트북 환경 최적화 (모바일에서도 기본 동작은 가능)
- 인터넷 연결 없이도 작동 가능
- 과제 제출용으로 단일 파일 구조(SPA 형태)로 구성됨
